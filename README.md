## 🧭 About Me

I'm a full-stack engineer who works across the whole stack — from database schema to API to the interface someone actually clicks on. I like owning a feature end to end instead of handing it off halfway.

Lately I'm spending most of my curiosity on **AI-powered applications** — LLM workflows, prompt engineering, and the tooling growing up around agents and RAG. I'm also pulled toward the product side of engineering: experimentation, analytics, and understanding *why* something works, not just whether it ships.
